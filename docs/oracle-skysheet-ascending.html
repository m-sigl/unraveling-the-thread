Skysheet Ascending

Filed under: Post-Corporate Prophecies

When the center hollows, only capital remains.

When jobs disappear, so do loyalties.

When no one has a stake, the stake bleeds out.

The machines will run the systems.

But no one will believe in them.

And belief is what made the system run.

So I left. I left to build something that breathes.
